# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - إصلاحات المهام وتطبيق نظام الإشعارات والتنظيف التلقائي (v-31)

### 2. الملفات المعدلة:
*   **[mockData.ts](file:///c:/Users/Admin/OneDrive/المستندات/NAMAA/src/data/mockData.ts)**:
    *   تحديث الأنواع في واجهة `Task` لتدعم الحالتين `'failed' | 'expired'` بداخل متغير `status`.
*   **[AppContext.tsx](file:///c:/Users/Admin/OneDrive/المستندات/NAMAA/src/context/AppContext.tsx)**:
    *   **مزامنة المهام:** تحديث دالة `mappedTasksFromDb` لتتوافق مع الأنواع الجديدة للحالة.
    *   **نظام الإشعارات السحابي:**
        *   جلب الإشعارات سحابياً من جدول `notifications` في Supabase عند التحميل (`fetchData`) مع وجود آلية تراجع وحفظ احتياطي محلي في `localStorage`.
        *   تحديث دالة `addNotification` لرفع الإشعارات لجدول `notifications` سحابياً وتخزينها محلياً.
        *   تحديث دالة `markNotificationAsRead` لتحديث حالة الإشعار سحابياً (`is_read: true`).
    *   **محفزات الإشعارات (Notification Triggers):**
        *   عند قيام الأب بتحويل مالي للطفل (`transferMoney`) -> إرسال إشعار للابن بتفاصيل المبلغ والسبب.
        *   عند قيام الابن بالتبرع (`addDonation`) -> إرسال إشعار للأب بمبلغ التبرع.
        *   عند المساهمة الاستثمارية بمشروع (`investInProject`) -> إرسال إشعار للأب بالتفاصيل.
        *   عند إيداع مبلغ في الحصالة (`addToGoal`) -> إرسال إشعار للأب بقيمة الإيداع.
        *   عند اكتمال أو بلوغ هدف الحصالة للمبلغ المطلوب (`addToGoal`) -> إرسال إشعار للابن: "مبروك! لقد حققت هدف الحصالة بنجاح 🎯".
        *   عند اعتماد الأب لمهمة الابن المنجزة (`finalizeTaskApproval`) -> إرسال إشعار للابن: "تم اعتماد مهمتك واستلام الجائزة!".
    *   **محرك التنظيف التلقائي (`runCleanup`):**
        *   حذف جميع المهام من جدول `kid_tasks` سحابياً حيث الحالات هي `approved` أو `failed` أو `expired`.
        *   حذف الإشعارات القديمة من جدول `notifications`.
        *   إعادة جلب وتزامن البيانات محلياً مع قاعدة البيانات عقب التنظيف.
*   **[KidTasksPage.tsx](file:///c:/Users/Admin/OneDrive/المستندات/NAMAA/src/pages/KidTasksPage.tsx)**:
    *   تعديل فلترة المهام في دالة `getTaskCategory` لتصنيف المهام ذات الحالة `failed` أو `expired` أو التي تجاوزت تاريخ النهاية (`endDate`) مع بقاء حالتها معلقة (`pending`) لتعرض بدقة وضمان عدم اختفائها من تبويب "غير منجزة".
*   **[KidDashboard.tsx](file:///c:/Users/Admin/OneDrive/المستندات/NAMAA/src/pages/KidDashboard.tsx)**:
    *   تطبيق تذكير المسؤولية المجتمعية كل ساعتين عبر مؤقت دوري `setInterval` مع إطلاقه كإشعار زجاجي منبثق بعد 5 ثوانٍ من تسجيل الدخول لتسهيل التحقق والتقييم.
*   **[DeveloperDashboard.tsx](file:///c:/Users/Admin/OneDrive/المستندات/NAMAA/src/pages/DeveloperDashboard.tsx)**:
