# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - تلميع نظام التنبيهات وإصلاح خيارات كارت المهام (v-29)

يوثق هذا السجل تطبيق نظام الإشعارات المتكامل (Notification System) عبر السياق العام والتخزين المحلي، وتكامل خيار المكافآت المخصصة في بطاقة التوصيات التفاعلية القابلة للتعديل (`SuggestedTaskWidget`).

## 📂 الملفات المشمولة بالتغيير

### 1. الملفات الجديدة:
*   **[v-29-notifications-polish-and-widget-fix.md](file:///d:/Documents/Namaa/docs/ai_sync/v-29-notifications-polish-and-widget-fix.md)**:
    *   سجل التزامن الحالي باللغة العربية لتوثيق التفاصيل البرمجية للتنبيهات والبطاقة التوليدية.

### 2. الملفات المعدلة:
*   **[SuggestedTaskWidget.tsx](file:///d:/Documents/Namaa/src/components/ui/SuggestedTaskWidget.tsx)**:
    *   إضافة الخيار المفقود **"مخصصة 🎁"** لقائمة نوع المكافأة.
    *   تطبيق إخفاء تلقائي لحقل المدخلات الرقمي للمبالغ عند اختيار المكافأة المخصصة، وعرض حقل نصي مخصص للهدية العينية (مثل: "ساعتين سوني 🎮").
    *   ربط واعتماد المدخلات المحدثة في دالة الإسناد `assignManualTask` بشكل دقيق.
*   **[AppContext.tsx](file:///d:/Documents/Namaa/src/context/AppContext.tsx)**:
    *   تعريف واجهة الإشعارات `Notification: { id, userId, role, title, message, createdAt, isRead }` وإضافتها للحالة العامة مع الحفظ والتحميل التلقائي من `localStorage`.
    *   تطبيق دالة `addNotification` و `markNotificationAsRead` لسهولة إدارة ومزامنة قراءة التنبيهات.
    *   **محفزات الإشعارات التلقائية (Triggers)**:
        1. **عند تعيين مهمة يدوياً** -> إرسال إشعار للابن: "قام والدك بتكليفك بمهمة جديدة".
        2. **عند تسليم الطفل إثبات المهمة** -> إرسال إشعار للأب: "أنهى ابنك مهمة بانتظار الاعتماد".
        3. **عند إنشاء مشروع استثماري عائلي** -> إرسال إشعار لكافة الأبناء: "مشروع استثماري عائلي جديد متاح!".
        4. **عند اكتمال تمويل المشروع الاستثماري** -> إرسال إشعار للأب: "تم اكتمال تمويل المشروع بنجاح!".
*   **[Topbar.tsx](file:///d:/Documents/Namaa/src/components/layout/Topbar.tsx)**:
    *   قراءة وتصفية التنبيهات من السياق العام وعرضها للمستخدم النشط فقط (تنبيهات ولي الأمر للوالد، وتنبيهات الابن لكل طفل بمفرده) مرتبة تنازلياً.
    *   تطبيق خطاف الإغلاق عند النقر بالخارج (Click Outside to Close) باستخدام `useRef` ومراقب الفأرة `mousedown`.
    *   تطبيق صياغة نصية نسبية للتواريخ (مثال: "الآن"، "قبل 5 دقائق"، "قبل ساعة") لتسهيل المتابعة.
