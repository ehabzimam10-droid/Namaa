# سجل تزامن الذكاء الاصطناعي (AI Sync Log) - ربط محرك Gemini وتوليد واجهات المهام الذكية (v-26)

يوثق هذا السجل تطبيق المرحلة الثالثة والأكثر حرجاً في بنية مستشار نماء الذكي (AI Coach): ربط المحادثة البرمجية الحية بواجهة Google Gemini API ومبدأ توليد الواجهات التفاعلية الديناميكية (Generative UI) لإسناد مهام الأبناء.

## 📂 الملفات المشمولة بالتغيير

### 1. الملفات الجديدة:
*   **[v-26-gemini-integration-gen-ui.md](file:///d:/Documents/Namaa/docs/ai_sync/v-26-gemini-integration-gen-ui.md)**:
    *   سجل التزامن الحالي لتوثيق تفاصيل عملية ربط المحرك السحابي لـ Gemini.
*   **[aiService.ts](file:///d:/Documents/Namaa/src/utils/aiService.ts)**:
    *   بناء خدمة الاتصال بـ Google Gemini API باستخدام حزمة `@google/generative-ai` ونموذج `gemini-1.5-flash`.
    *   تضمين موجه تعليمات صارم (System Prompt) لتجسيد هوية "مستشار بنك الإنماء المالي للأطفال" وتحليل بيانات الابن (الرصيد، أهداف الحصالة، المعاملات، ملاحظات الأب) وإرجاع كائن JSON دقيق يحتوي على: العنوان، قيمة المكافأة، نوعها، وسبب التوصية.
    *   تطبيق نمط المحاكاة الاحتياطي (Simulation Fallback) لضمان استقرار التطبيق كلياً وتقديم تجربة وظيفية متماسكة حتى في حال غياب أو عدم صحة مفتاح API.
*   **[SuggestedTaskWidget.tsx](file:///d:/Documents/Namaa/src/components/ui/SuggestedTaskWidget.tsx)**:
    *   كارت واجهة ديناميكي توليدي (Generative UI Widget) مبني بنمط زجاجي أنيق لعرض تفاصيل المهمة والسبب التربوي لاقتراحها.
    *   يحتوي على زرين تفاعليين: **"اعتماد المهمة 🎯"** و **"تعديل/رفض ❌"**؛ عند النقر على اعتماد، يتم استدعاء دالة `assignManualTask` من السياق وتثبيت حالة النجاح وإلغاء فاعلية الأزرار محلياً.

### 2. الملفات المعدلة:
*   **[FatherAICoachPage.tsx](file:///d:/Documents/Namaa/src/pages/FatherAICoachPage.tsx)**:
    *   تحديث هيكل مصفوفة الرسائل `Message` لتقبل كائنات React وعناصر ديناميكية (`React.ReactNode`) بدلاً من السلاسل النصية البسيطة.
    *   تكامل خدمة Gemini مع المحادثة: عند اختيار "اقتراح مهمة" للطفل، يظهر مؤشر تحميل مخصص، ثم يستدعي `suggestTaskForKid` مرسلاً بيانات الطفل ومفتاح API المخزن محلياً، ثم يدرج كائن `SuggestedTaskWidget` التوليدي كفقاعة محادثة تفاعلية داخل التطبيق.
